6.4 アロー関数
    アロー関数とは、無名関数の省略記法のこと
    6.4.1
        引数が無い場合  () => { 関数の本文; }
        引数が1つの場合　引数 => { 関数の本文; }
        引数が複数      (引数1, 引数2) => { 関数の本文; }
        関数の本文が1行の場合
            (引数1, 引数2) => 関数の本文;
        関数の本文が複数行の場合
            (引数1, 引数2) => {
                関数の本文;
                return 戻り値;
            }

            関数の本文が1行かつ戻り値がオブジェクトの場合
            (引数1, 引数2) => {(プロパティ1: 値1, プロパティ2:値2)}
